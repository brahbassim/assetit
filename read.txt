# IT Asset & License Management System

The IT Asset & License Management System is now complete. Here's a summary:

## Project Ready at C:\WAMP\www\assetit

## To Run:

```bash
cd C:\WAMP\www\assetit
php artisan serve
```

## Login Credentials:

- **Email:** admin@assetit.com
- **Password:** password

## Features Implemented:

- **Authentication** - Laravel UI with login/register
- **Role-based Access** - Spatie permissions (Admin, IT Manager, Technician, Viewer)
- **Complete CRUD** - Departments, Employees, Vendors, Categories, Hardware Assets, Software Licenses, License Assignments, Maintenance Records
- **Dashboard** - With Chart.js visualizations showing:
  - Assets by category/department
  - Licenses by vendor
  - License expiration timeline
  - License expiration alerts (90/30/7 days)
- **Reports Module** - PDF & Excel exports:
  - Hardware Inventory
  - License Utilization
  - Assets by Department
  - Maintenance Cost
  - License Expiration
- **Scheduled Command** - `php artisan licenses:check-expiration` (runs daily)
- **Audit Logging** - Automatic logging of create/update/delete/assign/revoke actions
- **SB Admin 2 UI** - Responsive Bootstrap 4 template with sidebar navigation

## Database:

- 5 departments
- 30 employees
- 10 vendors
- 8 asset categories
- 120 hardware assets
- 50 software licenses
- Random license assignments
- 80 maintenance records

## Additional Commands

```bash
# Run migrations and seed
php artisan migrate --seed

# Check license expirations
php artisan licenses:check-expiration

# Clear config cache
php artisan config:clear

# Clear route cache
php artisan route:clear
```
