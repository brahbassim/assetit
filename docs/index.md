## Overview

AssetIT is a powerful Laravel 12 application designed for IT departments to manage:

- **Hardware Assets** - Computers, servers, monitors, peripherals
- **Software Licenses** - With seat tracking and expiration alerts
- **Employees & Departments** - Organizational structure
- **Vendors** - Supplier relationships
- **Maintenance Records** - Service history and costs

## Key Features

- ✅ Role-Based Access Control (4 roles)
- ✅ 35+ granular permissions
- ✅ Multi-Language Support (EN, FR)
- ✅ Dashboard with Chart.js visualizations
- ✅ PDF and Excel export
- ✅ Automated license expiration alerts
- ✅ Complete audit logging
- ✅ Responsive Bootstrap 4 UI
